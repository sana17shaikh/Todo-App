## Overview
A simple, intuitive Todo List application designed to help you organize your tasks efficiently. This app allows users to add, delete, and mark tasks as completed, providing a seamless experience for managing daily to-dos.

## Features
- Add new tasks with a single click or pressing Enter
- Mark tasks as completed
- Delete individual tasks
- Clear all completed tasks
- Responsive design for desktop and mobile devices
- Persist tasks using local storage (if implemented)

## Technologies Used
- HTML5
- CSS3
- JavaScript
